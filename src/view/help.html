User help


General description

This window allows you to carry out maintenance operations basic Resource data. The user can through the available controls in the window: create, modify and delete the resources of the application. You can also generate a report that contains a list of resources with their corresponding data.


Operations


Create resources

To create a resource you must enter a tittle, link and a psychologist. Once the previous actions have been carried out, you must press the button Create to create the resource. If no error appears, the resource will be created and will appear in the table displayed in the window.


Modify resources data

To modify the data of a resource you must first select the resource's row by clicking on the desired resource of the table. The input fields of window will be informed with the data of the selected resource. These fields can be use to modify resouce data. Once the previous actions have been carried out, you must press the button Modify to finish it. If no error appears, the data will be modified and the updated data of the resource will appear in the table that shows in the window.


Delete resources

To delete a resource you must select the resource to delete by clicking on the desired row of the resource table. Then press the button Delete, and you will be prompted for confirmation to delete the resource. If you confirm the deletion and no error appears, the resource will be deleted and updated in the table displayed in the window.


Print the resource

Pushing the button Report a window will open where you can view a document which contains a list of the resources that exist in the application: Using the buttons at the top of the window you can print or store the report.


App Exit

Pushing Back you will appear in the welcome window. You can also click the "X" button to close the app.
